Project: FMCG Fulfillment Network Optimization
Mentor: Prof. Suvendu Naskar

Objective:
• Built a 36-day deployment schedule for a FMCG micro-fulfillment center targeting 8.5-minute delivery SLA goal
• Optimize resource allocation across 15 activities, reducing critical path duration to 31 days using leveling

Approach:
• Structured 15 WBS activities optimizing 2-minute picking, managing INR 2,000,000 initial CapEx efficiently
• Modeled CPM by 3-point PERT achieving 97.59% probability of 40-day success, crashed 5 activities to 31 days

Outcome:

• Achieved NPV of INR 29,754,156.90, IRR 143.19%, payback 1.5 years, ensuring strong returns overall
• Generated INR 4,712,000 cash flow Year 2, growing to INR 18,488,000; AOV INR 380–525 range steadily
